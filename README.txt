== Description ==

Permissions & Security Audit is a plugin that runs a series of tests to check common security issues with the following areas:

* File and folder permissions
* File and folder ownership
* Wordpress configuration settings
* Apache/PHP and hosting settings

Although this plugin will flag potential issues; it will not resolve these items. You will need to manually resolve them, use the help of a plugin or contact an individual who has the experience to help.
